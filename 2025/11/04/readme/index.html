第一次使用Hexo搭建博客，没有博客，全静态部署，有点不适应。
又一次折腾的开启，上一个博客在遭受攻击后，所以数据丢失了，WordPress太普及了，DDOS攻击很容易，VPS每个月的流量是限定的600G，竟然全给用完了，可恶的黑客，导致VPS被下线，和客服沟通后，在下个月流量恢复后，服务器重装才得以重新开启。当初是用加密货币购买的服务器，找了很多，支持加密货币的，且性价比合理的不多。
CN2直连服务器，国内网络访问速度非常好，看油管高清一点都不卡，访问X非常方便。
说回博客，没有后台，编辑文章使用的是MD格式，没有后台发布系统，需要学习下MD的基本用法，以前使用过markdown编辑器，非常方便，需要问下AI，哪个软件更合适。
本地编辑完还需要生成并推送到pages上。重要的是安全，纯静态，加载速度快。可以避免被攻击。
全新的体验，全新的开始，总要去走一走没去过的道路，看一看不一样的风景。
